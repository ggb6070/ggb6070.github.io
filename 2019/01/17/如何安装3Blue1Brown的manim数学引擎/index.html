manim入门到弃坑---安装

发表于 2019-01-17

参考自:
想学着制作3Blue1Brown风格的视频吗？先让manim跑起来再说~(不适用最新版本)
3Blue1Brown的manim！up主怒花1小时制作安装教程
动画库Manim学习笔记——安装与环境配置（一）
动画库manim[3b1b]的安装与配置
manim官方安装教程

一、前提

本教程仅适用于Windows系统

懂得使用cmd进入指定位置

运行cmd：使用快捷键Win+R，弹出运行对话框，输入cmd,回车即可。
进入指定位置：
例如：用cmd进入D:\3Blue1Brown\manim

确定操作系统的位数

如何查看电脑是多少位系统 32位和64位怎么看

安装Notepad++

使用manim会大量用到文本编辑器打开/修改.py文件，但使用Windows自带的记事本（文本编辑器的一种）会出现问题。
下载地址：https://notepad-plus-plus.org/

二、安装python3.7

截至目前python最新版本是Python 3.7.2
详细的python安装教程，参见：python安装教程（Windows系统,python3.7为例）
注意：一定要勾选”Add Python 3.7 to PATH”,使其添加到windows系统环境变量；检查是否安装成功

三、安装manim源码

下载源码

官方源码：https://github.com/3b1b/manim/
第三方源码（推荐）：https://github.com/cc3630/manim
这个版本的源码是经过修改的，出现的问题会比较少。

方法一（推荐）：
点击”Clone or Download”
选择”Download ZIP”
解压即可
方法二：
使用git安装
git clone https://github.com/3b1b/manim.git

修改源码

用Notepad++打开源码根目录下的requirements.txt，删掉其中numpy和scipy所在的行，后面单独安装，否则可能发生错误：

查看格式

参考自：https://blog.csdn.net/nnoner/article/details/84997202
查看python对应的pip所支持的文件格式
运行cmd依次输入:
python
import pip._internal
print(pip._internal.pep425tags.get_supported())
结果如下

说明第四步骤要安装cp37-cp37m-win_amd64版本的numpy和scipy

四、安装numpy和scipy

下载

numpy下载地址：http://www.lfd.uci.edu/~gohlke/pythonlibs/#numpy
scipy下载地址：http://www.lfd.uci.edu/~gohlke/pythonlibs/#scipy
选择正确的版本下载（可选择最新版本或requirements.txt里对应的版本）
根据('cp37', 'cp37m', 'win_amd64')可知应选择下载
numpy-1.16.0+mkl-cp37-cp37m-win_amd64.whl
scipy-1.2.0-cp37-cp37m-win_amd64.whl
否则出现错误
numpy-1.16.0+mkl-cp37-cp37m-win32.whl is not a supported wheel on this platform.

安装

运行cmd
先安装numpy模块
pip install D:\3Blue1Brown\manim-master\numpy-1.16.0+mkl-cp37-cp37m-win_amd64.whl

参考https://blog.csdn.net/wtwcsdn123/article/details/81623451
再安装scipy模块
pip install D:\3Blue1Brown\manim-master\scipy-1.2.0-cp37-cp37m-win_amd64.whl

五、安装其他模块

运行cmd，cd进入manim-master目录
python -m pip install -r requirements.txt
自动安装剩余的模块

六、安装ffmpeg和dvisvgm

ffmpeg安装参见：https://blog.csdn.net/a18852867035/article/details/82053611
dvisvgm下载地址：https://dvisvgm.de/Downloads/
将两者添加到环境变量，参见：想学着制作3Blue1Brown风格的视频吗？先让manim跑起来再说~第五步第3点环境变量的设置

七、安装miktex

下载地址：https://miktex.org/download
安装参见：https://jingyan.baidu.com/article/48b558e3ffa3667f38c09a99.html
其中，选择 “Ask me first”，在以后的使用中如果缺失相关组件，会提醒你安装，也可选择不通知自动安装。如需支持中文，参见：动画库Manim学习笔记——安装与环境配置（一）第四步

八、检查

运行cmd,输入：

如果有其中一条命令没有正常显示版本信息，则检查相应的安装/配置过程是否有误。

九、运行

运行cmd，cd进入manim-master目录
python -m manim example_scenes.py -pl

输入相应的数字，可以渲染出相应场景的视频，自动保存在manim-master目录下会自动新建的video文件夹中
